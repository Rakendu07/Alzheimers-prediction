# Alzheimer's Disease Prediction Using MRI Scans

## Overview
This project uses deep learning to classify Alzheimer's disease stages from brain MRI images into four categories:
- Non-Demented
- Very Mild Dementia
- Mild Dementia
- Moderate Dementia

## Features
- 4-class classification
- Transfer learning (ResNet50/VGG16/EfficientNet)
- Grad-CAM visualization
- Single-image prediction
- Complete Colab notebook

## Technologies Used
Python • TensorFlow/Keras • OpenCV • NumPy • Pandas • Matplotlib • scikit-learn
